# machine-learning-projects
A collection of my machine learning projects, experiments, and learning progress.
All projects are implemented primarily in **Python** using **Google Colab** and various AI assistants (ChatGPT 5.1 and Gemini 3.0) for ideation, debugging, and documentation.

---

## Projects

1. **Meat Freshness Classifier using Convolutional Neural Networks**
   - **Tech:** Python, scikit-learn, Tensorflow/keras, NumPy, Pandas, Matplotlib
   - **Description:** Using Convolutional Neural Networks (CNN) to classify meat freshness as Fresh, half-fresh and not fresh.
   - **Folder:** `Meat Freshness Classifier/`

2. **Mental Health Sentiment Analysis**
   - **Tech:** Python, scikit-learn, Tensorflow/keras, NumPy, Pandas, Matplotlib, NLTK, spacy, nplaug
   - **Description:** Using NLP to classify mental health statements as depressed (1) or normal (0).
   - **Folder:** `Mental Health Sentiment Analysis/`

3. **Weather Classifier**
   - **Tech:** Python, scikit-learn, NumPy, Pandas, Matplotlib
   - **Description:** Applying logistic regression and random forest classifier to categorise meteorological data into normal and extreme classes.
   - **Folder:** `Weather Classifer/`
